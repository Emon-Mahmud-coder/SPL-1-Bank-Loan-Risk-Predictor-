# Loan Risk Prediction System (SPL-1)

This project is a high-performance C++ implementation of a **Random Forest algorithm** designed to predict loan default risks. It analyzes applicant data such as credit scores, income, and debt-to-income ratios to provide a risk assessment and banking recommendations.
